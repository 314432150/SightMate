# 准星助手

一个简单、好用的安卓准星辅助工具。

## 功能特点

- 可调节大小的准星点
- 自定义准星颜色（红、绿、蓝、黄、白、青）
- 可调节透明度
- 方向键精确调整准星位置
- 保存准星设置和位置

## 系统要求

- Android 11.0 (API 30) 及以上
- 需要悬浮窗权限

## 如何使用

1. 安装应用后首次运行需要授予悬浮窗权限
2. 打开准星显示开关
3. 通过样式页面调整准星大小、颜色和透明度
4. 使用位置调节功能微调准星位置

## 关于
该项目使用 Kotlin 和 Jetpack Compose 开发，采用现代 Android 开发实践。
